---

### 값 타입과 참조 타입

솔리디티의 **자료형**은 **값 타입**(Value type)과 **참조 타입**(Reference type)으로 나뉜다.

- **[값 타입]** : 값이 할당되거나 함수 인자로 활용되면 해당 값 자체가 복사
                    
- **[참조 타입]** : 현재 해당하는 값의 주소만 복사
                    - 데이터 저장 위치를 명시해야 함
                    - 배열, 매핑, 구조체로 구성됨

---

### 불리언(Boolean) 타입 : bool

- 참과 거짓으로 이루어진 자료형
- true / false

---

### 정수(Integer) 타입 : int와 uint

- 솔리디티는 소수점이 있는 숫자를 표현하는 float나 double 지원 X 

---

### 바이트 타입 : bytes

- **[고정 크기 바이트 배열]**
    - 값 타입
    - 사용할 바이트 미리 지정
    - bytes1~bytes32

- **[동적 크기 바이트 배열]**
    - 참조 타입
    - 사용할 바이트 미리 지정 X
    - bytes
